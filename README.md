# Brand Shop — Интернет-магазин на Next.js

Адаптивный интернет-магазин с фильтрами, пагинацией и корзиной.

## Демо

**Сайт:** https://brand-shop-avion.vercel.app/

## Стек

- **Frontend:** Next.js 14, TypeScript
- **Стилизация:** SCSS (БЭМ)

## Функционал

- Каталог товаров с пагинацией
- Фильтры по категориям, цене, бренду
- Поиск по названию
- Корзина (добавление/удаление товаров)
- Адаптивная верстка (computer-first)
- Оптимизация изображений через Next.js Image
